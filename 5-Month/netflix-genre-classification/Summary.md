# Netflix Content Analysis & Genre Classification 

## Overview
This project analyzes the Netflix Movies & TV Shows dataset and builds a machine learning model to predict the **primary genre** of a title using its metadata (title, description, cast, director, country).

The project follows a complete **end-to-end data science workflow**:
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- NLP-based feature engineering (TF-IDF)
- Genre classification model
- Model evaluation
- Streamlit app deployment

## Dataset
- Source: Netflix Movies and TV Shows dataset (Kaggle)
- Key columns: title, director, cast, country, date_added, release_year, rating, duration, listed_in (genres), description
